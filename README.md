# Front-Catamiki

Це фронтенд-проєкт, створений з використанням технологій React та TypeScript. Структура проєкту побудована за принципами FSD (Feature-Sliced Design), що дозволяє розділяти код за функціональними блоками та підтримувати його у зрозумілому стані при масштабуванні.

## Використані технології

- **React** – бібліотека для побудови користувацького інтерфейсу.
- **TypeScript** – для забезпечення типізації та підвищення надійності коду.
- **Vite** – сучасний інструмент для швидкого запуску проєкту та розробки.
- **FSD (Feature-Sliced Design)** – підхід до організації коду за функціональними блоками:
  - Кожна фіча або модуль містить свої компоненти, утиліти та стилі.
  - Це дозволяє легше підтримувати та масштабувати проєкт.

## Структура проєкту

- `src/` – основний код проєкту.
  - `app/` – загальні налаштування додатку (точка входу, роутинг).
  - `entities/` – основні сутності додатку.
  - `features/` – реалізація функціональних блоків.
- `public/` – статичні файли (картинки).

## Мета проєкту

Проєкт створено як частину курсової роботи з метою демонстрації навичок розробки фронтенду з використанням сучасних технологій та організації коду за принципами FSD.
